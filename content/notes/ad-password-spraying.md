## Overview
Password spraying checks whether many accounts accept a small number of weak passwords without triggering effective detection or lockout controls.

## Steps
- Get explicit written approval for spraying.
- Confirm lockout threshold and observation window.
- Use a small candidate set and one password per window.
- Monitor for lockouts and stop if accounts are affected.
- Validate any success carefully and document the control gap.

## Evidence
Record target domain, test window, account count, password count, successful accounts, and lockout observations.
