## Overview
Frida runtime hooking lets you observe and modify app behavior while it runs on a test device.

## Steps
- Confirm device, app, and build are in scope.
- Start Frida server or use Frida gadget.
- Identify the package and process.
- Hook targeted methods instead of broad random patching.
- Capture evidence before and after the hook.

## Evidence
Save hook script, method name, app state, bypassed control, and screenshots or logs.
