## Overview
API endpoint risk register reporting lists endpoints, methods, data classes, authorization state, and findings in one place.

## Steps
- Import paths from OpenAPI where available.
- Record method, auth requirement, object type, and data class.
- Map findings to OWASP API categories.
- Include evidence request and response references.
- Add retest cases per endpoint.

## Evidence
Capture endpoint, method, role, object id, response code, finding, and remediation status.
