## Overview
File upload testing checks whether users can upload unsafe content, bypass validation, or access files outside the intended audience.

## Steps
- Identify upload limits and allowed extensions.
- Test MIME type, extension, magic bytes, and filename handling.
- Check storage location and public access.
- Try harmless metadata and polyglot-style validation checks.
- Avoid active payloads unless explicitly authorized.

## Evidence
Record uploaded filename, validation bypass, storage URL, access control, and impact.
