## Overview
Firewall rule persistence review checks whether host or perimeter firewall changes create long-lived access paths.

## Steps
- Confirm firewall review is in scope.
- Export active host and network firewall rules.
- Identify new allow rules, broad source ranges, and unusual ports.
- Map rules to listening services, owners, and business need.
- Document cleanup and monitoring recommendations.

## Evidence
Record rule name, source, destination, port, action, creation or modification time, owner, and remediation status.
