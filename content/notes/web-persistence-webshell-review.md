## Overview
Web shell persistence review checks whether executable files in web roots or upload paths could maintain server-side access.

## Steps
- Confirm server-side review is authorized.
- Search recent and unusual web-root files.
- Review upload directories for executable extensions.
- Correlate timestamps with testing activity.
- Document safe cleanup and monitoring guidance.

## Evidence
Record file path, timestamp, owner, suspicious function, request path, and cleanup decision.
