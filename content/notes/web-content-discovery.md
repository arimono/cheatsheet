## Overview
Content discovery finds hidden routes, backup files, admin panels, and technology hints on web targets.

## Steps
- Check robots.txt, sitemap.xml, and JavaScript files.
- Run wordlists with a safe request rate.
- Review interesting redirects, forbidden paths, and unusual sizes.
- Test discovered endpoints manually before escalating.
- Link endpoints to auth, injection, and upload notes.

## Evidence
Record URL, status code, content length, authentication requirement, and why the path matters.
